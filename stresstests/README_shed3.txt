Программа shed3.py
работает на python версии 2.4

Программа отправляет файлы из каталога по FTP протоколу.
Алгоритм работы:
читает файл start.txt в котором:
строка №1 - флаг остановки прграммы, если не START то завершить программу
строка №2 - дата и время когда начинаем отправку файлов по FTP
строка №3 - время в секундах между отправками файлов

Если цивра в строке №3 равна 5 то будет слеана пуза в 5 секунд между отправками файлов.
